# Sistema de Controle de Estoque para Farmácia

## Sobre o Projeto

Este projeto foi desenvolvido em Python como parte do Programa Integrado Multidisciplinar (PIM) da UNIP, com o objetivo de aplicar na prática conceitos de lógica de programação e desenvolvimento de sistemas.

O sistema simula o gerenciamento de estoque de uma farmácia, permitindo o controle básico de produtos e quantidades, reproduzindo situações comuns encontradas em ambientes de gestão.

## Funcionalidades

- Cadastro de medicamentos
- Consulta de produtos cadastrados
- Controle de estoque
- Atualização de quantidades
- Exibição das informações armazenadas

## Tecnologias Utilizadas

- Python

## Objetivos do Projeto

- Aplicar conceitos de lógica de programação
- Desenvolver habilidades de resolução de problemas
- Praticar estruturas condicionais e de repetição
- Simular um cenário real de controle de estoque
